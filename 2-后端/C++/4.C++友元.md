# 全局函数做友元
### 作用：全局函数可以访问私有的成员对象。
### 语法：`friend 返回属性类型 全局函数名()`
```cpp
class Person {
	// 告诉编译器test全局函数为友元，可以访问私有成员对象
	friend void test();

public:
	string phone;
	Person() {
		phone = "Iphone";
		bag = "爱马仕";
	};

private:
	string bag;
};

// 全局函数
void test() {
	Person p;
	cout << "访问手机： " << p.phone << endl;
	cout << "访问包： " << p.bag << endl;
};

void main() {
	test();
	system("pause");
};
```

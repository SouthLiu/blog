# stack(堆栈)容器
### stack是一种**先进后出**的数据结构，类似一个桶，往桶里放东西，只能拿出来最后放的那个东西。

## 常用接口
* `stack<T> stk;` // stack采用模板类实现，stacK对象的默认构造形式
* `stack(const stack &stk);` // 拷贝构造函数

### 赋值操作:
* `stack& operator = (const stack &stk);` // 重载等号操作符

### 数据存储：
* `push(elem);` // 向栈顶添加元素
* `pop();` // 从栈顶删除
* `top();` // 返回栈的大小

### 大小操作：
* `empty();` // 判断堆栈是否为空
* `size();` // 返回栈的大小

```cpp
#include <iostream>
using namespace std;
#include <stack>

int main() {
	stack<int>stk;
	stk.push(10);
	stk.push(20);
	stk.push(30);

	while (!stk.empty()) {
		cout << stk.top() << endl; // 30 20 10
		stk.pop();
	}

	return 0;
};
```

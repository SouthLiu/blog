# 3.编写一个程序，找到2000年到3200年(包括以内)中所有可被7整除但不能被5整除的数字，将结果打印在一行中，以逗号分隔。
```python
arr = []

for i in range(2000, 3200):
  if (i % 7 == 0) and (i / 5 != 5):
    arr.append(str(i))

print(','.join(arr))
```

# 4.编写一个可以计算给定数阶乘的程序，结果以逗号分隔，打印在一行上。
```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
    
n = int(input("请输入一个整数："))
result = factorial(n)
print(','.join(str(result)))
```

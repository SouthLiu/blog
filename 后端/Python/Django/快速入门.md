# 常用命令
生成项目
```python
django-admin startproject name
```
生成应用
```python
python manage.py startapp name
```
启动项目
```python
python manage.py runserver
python3 manage.py runserver 0.0.0.0:8002 
```
创建数据库
```python
python manage.py makemigrations
python manage.py migrate
```

# 变量
```
a = 5
b = 6
a, b = b, a
print(a) # 6
print(b) # 5
print(f"a是{a},b是{b}") # a是6,b是5
```

# 数据类型转型
* int() 转整形
* float() 转浮点型
* str() 转字符串

# 检查类型
```
print(type(1)) # <class 'int'>
```

# 运算符差异
* / 浮点型
* // 整除
```
print(5 / 2)  # 2.5
print(5 // 2)  # 2
```

# 逻辑运算符
```
print(1 and 0)  # 0
print(1 or 0)  # 1
print(not 0)  # True
```

# 字符串
三引号支持**换行**
```
print("""I
Am
South""")

# I
# Am
# South
```

# 切片
语法：序列[开始位置下标:结束位置下标:步长]
```
name = "我是新手程序员"
print(name[2:5])  # 新手程
print(name[2:5:2])  # 新程
print(name[::-1])  # 员序程手新是我
print(name[-4:-1])  # 手程序
```

# 字符串方法
* startswith 检测字符串是否以子串开头
* endswith 检测字符串是否以子串结尾
```
name = "我是新手程序员"
print(name.startswith("我是"))  # True
print(name.startswith("你是"))  # False
print(name.endswith("程序员"))  # True
```

# 字符串转ASCII码
```python
print(ord('北')) # 21271
print(ord('京')) # 20140
print(chr(21271), chr(20140)) # 北 京
```

| 函数 | 描述说明 |
| --- | --- |
| int(x) | 将x转为整数类型 |
| float(x) | 将x转为浮点数类型 |
| str(x) | 将x转为字符串 |
| chr(x) | 将一个整数x转为一个字符 |
| ord(x) | 将一个整数x转为其对应的整数值 |
| hex(x) | 将一个整数x转为一个十六进制字符串 |
| oct(x) | 将一个整数x转为一个八进制字符串 |
| bin(x) | 将一个整数x转为一个二进制字符串 |
